# GENX time series variability analyses

## Summary

This repository contains code to analyze how warming altered the variability of methane emissions in the Greenhouse gas Emissions NeXus ([GENX](https://serc.si.edu/labs/biogeochemistry-projects/genx)) experiment, using data from March–November 2025. GENX is located within the Global Change Research Wetland ([GCReW](https://serc.si.edu/gcrew)) in Edgewater, MD, USA. For more information, please contact Abigail Lewis ([abigail.sl.lewis@gmail.com](abigail.sl.lewis@gmail.com)) or Genevieve Noyce ([NoyceG@si.edu](mailto:NoyceG@si.edu)).
